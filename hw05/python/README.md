## Instructions
### Usage
`python3 main.py [--language LANGUAGE] [--n_suggestions N_SUGGESTIONS]`

### Optional arguments
```
--language LANGUAGE             Language of input text.
--n_suggestions N_SUGGESTIONS   Maximum number of corrections to suggest.
```

## Used libraries
* `nltk`;
* `numpy`;
* `hunspell`;
* `pyphonetics`;
* `jaro-winkler`;
* `textdistance`;
* `wordfreq`.
